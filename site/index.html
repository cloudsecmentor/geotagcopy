Workflow

How it works

  1. Choose two folders. Select one folder with geotagged phone photos and another with untagged camera photos.
  2. Review timestamp matches. GeoTagCopy reads EXIF metadata and groups likely matches by GPS location.
  3. Apply GPS tags. Select the matches you trust, then write GPS coordinates with ExifTool.
