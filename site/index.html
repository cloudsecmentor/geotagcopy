Workflow

How to copy GPS from phone photos to camera photos

  1. Choose your geotagged phone folder. Use iPhone, Android, or other photos that already contain GPS metadata.
  2. Choose your camera folder. Select the DSLR or mirrorless photos that need location data.
  3. Review timestamp matches. GeoTagCopy reads EXIF metadata and groups likely matches by GPS location.
  4. Apply GPS tags locally. Select the matches you trust, then write coordinates with ExifTool.

FAQ

Common geotagging questions

Can I geotag Sony, Canon, Nikon, or Fujifilm photos after the shoot?

Yes, as long as you have phone photos with GPS metadata from the same timeframe. GeoTagCopy uses those files as location references.

Is this a replacement for Lightroom Map or Apple Photos maps?

It complements them. GeoTagCopy writes GPS coordinates first, then apps like Lightroom and Photos can use that metadata for map views, filtering, and organization.

What makes it different from running ExifTool commands?

ExifTool does the metadata writing, but GeoTagCopy gives you a desktop review flow so you can inspect matches before changing files.
